# APEI Brand Colors

## Primary Palette

### APEI Navy

The primary brand color. Use for title slide backgrounds, closing slides, section divider backgrounds, body headline text on light backgrounds, and the upper-right stacked corner-mark triangles.

| Format | Value |
|--------|-------|
| Hex | `#22314E` |
| RGB | 34 / 49 / 78 |
| CMYK | 94 / 73 / 5 / 69 |
| Pantone Coated | 533 |

### APEI Green

Sharp accent. Use for corner-mark triangles, agenda numbering, short underline accents beneath titles, key data highlights, and callout bars. Use sparingly — it's meant to punctuate, not fill.

| Format | Value |
|--------|-------|
| Hex | `#CCDC00` |
| RGB | 204 / 220 / 0 |
| CMYK | 25 / 0 / 100 / 0 |
| Pantone Coated | 389 |

### APEI Gray

Neutral. Use for dividers, light section backgrounds, de-emphasized text, alternating table rows, and subtle fills.

| Format | Value |
|--------|-------|
| Hex | `#E5E5E5` |
| RGB | 229 / 229 / 229 |
| CMYK | 9 / 6 / 7 / 0 |

### APEI Blue

Secondary accent. Use for charts, data visualizations, map highlights, and infographic slides alongside navy and green.

| Format | Value |
|--------|-------|
| Hex | `#0072CF` |
| RGB | 0 / 114 / 207 |
| CMYK | 90 / 47 / 0 / 0 |

## Extended Palette

When a chart, infographic, or diagram needs more than the four primaries, use these complementary tones. They're chosen to harmonize with APEI Navy, Green, and Blue.

| Purpose | Name | Hex |
|---------|------|--------|
| Darker navy shade | Deep Navy | `#0F1A2E` |
| Mid-navy (charts) | Steel Blue | `#3E5373` |
| Light navy (backgrounds) | Mist | `#E8ECF3` |
| Warm gray (muted text) | Slate Gray | `#6B7280` |
| Chart neutral | Chart Gray | `#B8BEC7` |
| Soft green (data rows) | Muted Lime | `#E8EE99` |
| White | White | `#FFFFFF` |
| Near-black (body text) | Near Black | `#1A1A1A` |

## Rules

- Never introduce a new hue (red, orange, purple, teal) without an explicit reason and stakeholder approval.
- Tints and shades of the primary palette are always safe.
- If a warning or error tone is needed, ask before using red — APEI decks generally avoid alarm colors.
- One color should dominate each slide (60–70% visual weight). Never give all four primaries equal weight on the same slide.

## Typography

All APEI text uses **Open Sans**. The template uses three weights:

| Weight | PPTX typeface string | Use |
|--------|---------------------|------|
| Light | `Open Sans Light` | Large titles, hero numbers |
| Regular | `Open Sans` | Body text, captions |
| Bold | `Open Sans` + `b="1"` | Section eyebrows, table headers, emphasis |

Never swap to Calibri, Arial, or Helvetica on APEI decks.
